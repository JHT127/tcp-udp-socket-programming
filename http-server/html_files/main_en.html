Computer Networks: Protocol Layers

Computer network protocols are organized into layers to manage complexity and provide modularity.

OSI Reference Model

  1. Physical Layer: Transmits raw bit stream over physical medium
  2. Data Link Layer: Provides node-to-node data transfer
  3. Network Layer: Routes packets across multiple networks
  4. Transport Layer: Ensures reliable data transfer
  5. Session Layer: Manages communication sessions
  6. Presentation Layer: Translates data between formats
  7. Application Layer: Provides network services to applications
